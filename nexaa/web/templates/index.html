Qué significa esto
  • URL no accesible: el sitio rechazó la conexión (404, 5xx, timeout, DNS).
  • robots.txt bloquea: el sitio prohíbe el scraping para nuestro bot.
  • Sitio JS pesado: la página no devuelve el artículo en HTML, solo JS (no podemos ejecutarlo).
  • Rate limit: el proveedor de IA llegó a su límite. Esperá unos segundos.
  • No hay API key: revisá .env y reiniciá el servidor.
